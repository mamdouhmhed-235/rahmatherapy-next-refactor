Visual audit checks:
1. Does the page feel premium, trustworthy, warm, and professional?
2. Does it use the intended design system consistently?
3. Are cards, images, overlays, dark sections, CTAs, and spacing coherent?
4. Does visual hierarchy guide users toward booking or learning more?
5. Is any section cluttered, generic, weak, cramped, messy, or visually unappealing?
6. Are placeholder images professional and clearly labelled?
7. Do services package cards visually match planned homepage package cards?
8. Do package pages feel sales-led and specific rather than generic?
9. Is the reviews wall controlled and usable rather than chaotic?
10. Are FAQ/aftercare tabs and About timeline obvious to use?
11. Does the page avoid generic, visually weak, cramped, misaligned, or unprofessional patterns?
12. Does each page meet the frontend standard expected from a polished production wellness site?

Design consistency matrix:
Create a matrix in 03-visual-ui-ux-audit.md comparing:
- hero treatment
- card radius
- spacing rhythm
- typography scale
- image overlay contrast
- CTA styling
- dark green sections
- final CTA treatment
- placeholder quality
- issue IDs

- 390px mobile
- 768px tablet
- 1024px small desktop
- 1440px desktop

Responsive checks:
1. No horizontal overflow.
2. Text fits containers.
3. CTAs wrap gracefully.
4. Cards stack cleanly.
5. Header/nav remains usable.
6. Image overlays remain readable.
7. Review wall remains usable.
8. Tables or comparison content do not break mobile.
9. Tabs/filters are usable on touch.
10. Final CTA sections remain clear.

Write findings into:
- 03-visual-ui-ux-audit.md
- 04-responsive-mobile-audit.md
- 09-master-issue-register.md

- UX
- RESP

- No screenshots committed.
