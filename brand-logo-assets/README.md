# Rahma Therapy Logo Assets

Brand asset pack regenerated from `RahmaTherapy_Logo.jpg` on 2026-04-25.

## Folders

- `raster-exact/` - primary fidelity reference. These assets preserve the supplied JPG appearance, with cleaned transparent logo PNGs and SVG wrappers for SVG-named files.
- `vector-trace/` - automated colour-separated SVG trace of the full supplied logo, including the service line.
- `vector-trace-no-tagline/` - automated colour-separated SVG trace with the bottom service line removed.
- `archive-2026-04-25-original/` - previous root-level logo assets kept for rollback/reference.

## Files In Each Generated Pack

- `logo-refined.svg` / `logo-refined.png` - main logo.
- `logo-horizontal.svg` / `logo-horizontal.png` - horizontal lockup.
- `logo-trace.svg` / `logo-trace.png` - trace/reference variant.
- `logo-mark.svg` / `logo-mark.png` - compact `R` mark.
- `logo-monochrome.svg` / `logo-monochrome.png` - one-colour version.
- `favicon.svg`, `favicon.ico`, `favicon-32.png`, `favicon-16.png` - favicon assets derived from the `R` mark.
- `apple-touch-icon.png` - iOS home screen icon derived from the `R` mark.
- `social-preview.png` - 1200x630 sharing preview.

## Usage Notes

- Use `raster-exact/` when matching the supplied logo matters most.
- Use `vector-trace/` when a path-based SVG is required and the full logo should include the service line.
- Use `vector-trace-no-tagline/` when the bottom service line should be omitted.
- The app was not rewired as part of this asset update; these files are standalone brand assets only.
